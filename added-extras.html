How to use

All icons require an <i> tag with a unique class, prefixed with icon-. To use, place the following code just about anywhere:

  1. <i class="icon-search"></i>

There are also styles available for inverted (white) icons, made ready with one extra class. We will specifically enforce this class on hover and active states for nav and dropdown links.

  1. <i class="icon-search icon-white"></i>

Heads up! When using beside strings of text, as in buttons or nav links, be sure to leave a space after the <i> tag for proper spacing.

Now, here's a look at a typical HTML file:

<!DOCTYPE html>
<html>
  <head>
    <title>Bootstrap 101 Template</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
  </body>
</html>

To make this a Bootstrapped template, just include the appropriate CSS and JS files:

<!DOCTYPE html>
<html>
  <head>
    <title>Bootstrap 101 Template</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet" media="screen">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="http://code.jquery.com/jquery.js"></script>
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

And you're set! With those two files added, you can begin to develop any site or application with Bootstrap.

Font Awesome Animations

Setup

  1. <link rel="stylesheet" href="assets/css/icon-hover.css">
  2. <!-- This file is required for slide and flip effects only -->
  3. <link rel="stylesheet" href="assets/css/icon-hover-icons.css">

Slide, flip, and colored brands

Add .icon-hover-slide, .icon-hover-flip, or .icon-hover-color (colored brands) to the icon tag. .icon-hover-color can be combined with slide or flip.

Slide and flip effects support Chrome, Firefox, Opera 15+, and IE10+. Flip only supports Chrome with GPU acceleration enabled.

Credit to draebb for writing this code.

icon-hover-slide icon-hover-flip icon-hover-color icon-hover-color icon-hover-slide icon-hover-color icon-hover-flip 
  1. <div class="links-icon-text">
  2.   <a href="#"><i class="icon-smile icon-hover-slide"></i>icon-hover-slide</a>
  3.   <a href="#"><i class="icon-smile icon-hover-flip"></i>icon-hover-flip</a>
  4.   <a href="#"><i class="icon-google-plus icon-hover-color"></i>icon-hover-color</a>
  5.   <a href="#"><i class="icon-twitter icon-hover-color icon-hover-slide"></i>icon-hover-color icon-hover-slide</a>
  6.   <a href="#"><i class="icon-facebook-sign icon-hover-color icon-hover-flip"></i>icon-hover-color icon-hover-flip</a>
  7. </div>
  8.  
  9. <!-- .icon-* classes can be added to <a> directly for links without text -->
  10. <div class="links-icon">
  11.   <a href="#" class="icon-smile icon-hover-color"></a>
  12.   <a href="#" class="icon-dribbble icon-hover-color icon-hover-slide"></a>
  13.   <a href="#" class="icon-github icon-hover-color icon-hover-flip"></a>
  14.   <!-- Use aria-label or .visuallyhidden to make links accessible for screen readers -->
  15.   <a aria-label="Text" href="#" class="..."></a>
  16.   <a href="#" class="..."><span class="visuallyhidden">Text</span></a>
  17. </div>

CSS example:

  1. .links-icon-text a {
  2.   display: block;
  3.   padding: 3px 0;
  4. }
  5. .links-icon-text a > i {
  6.   color: #999;
  7.   font-size: 20px;
  8.   vertical-align: -10%;
  9.   width: 1.2em;
  10. }
  11. .links-icon-text a:hover > i {
  12.   color: #555;
  13. }
  14.  
  15. .links-icon a {
  16.   color: #999;
  17.   font-size: 28px;
  18.   margin-right: 5px;
  19. }
  20. .links-icon a:hover {
  21.   color: #555;
  22. }

If .icon-hover-color is used alone (no slide or flip), you may add it to all icons even those were not listed above. It set color transition, display: inline-block, and removes underlines.

<script type="text/javascript">
$(document).ready(function(){
$.cookieBar({  });
});
</script>
